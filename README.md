# 多资产投资组合VaR风险分析系统

## 项目概述
这是一个完整的Value-at-Risk (VaR) 分析系统，用于多资产投资组合的风险度量。项目实现了三种主要的VaR计算方法，并提供了丰富的可视化和报告功能。

## 功能特性
- ✅ 历史模拟法VaR计算
- ✅ 参数法（方差-协方差）VaR计算  
- ✅ 蒙特卡洛模拟法VaR计算
- ✅ Expected Shortfall (CVaR) 计算
- ✅ 交互式可视化仪表板
- ✅ 自动报告生成（文本和LaTeX格式）
- ✅ 单元测试覆盖
